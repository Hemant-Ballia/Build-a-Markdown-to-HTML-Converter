# 🚀 Build-a-Markdown-to-HTML-Converter

A lightweight, real-time web application to transform Markdown syntax into clean HTML code and live previews. Perfect for developers and writers who want a fast distraction-free editing experience.

## ✨ Features
- **Real-time Conversion:** Instant transformation as you type using optimized Regex.
- **Triple Pane Interface:** - **Editor:** For writing Markdown.
    - **HTML Output:** To see the raw generated HTML code.
    - **Live Preview:** To see how it looks visually.
- **Regex-Powered:** Custom-built logic for:
    - Headings (H1, H2, H3)
    - Bold and Italics
    - Blockquotes
    - Links and Images
- **Modern UI:** Clean, responsive design using CSS Grid and dark/light accents.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Grid & Flexbox)
- **Logic:** Vanilla JavaScript (ES6+ Regular Expressions)
